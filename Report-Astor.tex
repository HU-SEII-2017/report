\begin{abstract}
  Context, Problematic, Solution

Astor is a automatic, multidisciplinary testing and repair library for Java. It contains three different modi, GenProg, Kali and MutRepair. With focussing on the GenProg modus we found out that Astor's modus it may have a problem with enums, static variables that do not change their state. We ran Astor in its original version and a bug fixed version on Dataset4J, a set of real-life bugs in Java, and we will compare both results.
